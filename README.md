# Personalized Course Recommendation System

## Overview
An AI-assisted web application built using Lovable that recommends personalized courses based on users' skills, interests, education level, and career goals.

## Features
- Personalized course recommendations
- Learning roadmaps
- Course search and filtering
- Saved courses
- Progress tracking

## Tech Stack
- Lovable
- React
- TypeScript
- Tailwind CSS
